### Colors
The website uses CSS custom properties for easy color customization. Edit the `:root` section in `assets/css/main.css`:

```css
:root {
  --primary-color: #3b82f6;
  --secondary-color: #6366f1;
  --accent-color: #f59e0b;
  /* ... other colors */
}
```
